Здравстуй
Это обычная консольная программа работы с заметками в форматах: json, csv
(При желании можно будет с легкостью добавить какой нибудь другой формат)

Чтобы запустить её нужно запусть файл Button.py
- Далее запускается меню, где нам предложит программа либо создать новую заметку(в выбранном нами формате файла), 
посмотреть список заметок по выбранному нами формату файла, либо выйти из программы

- Заметки создаются в новой папке в той же директории что и папка с программой, с пометкой формата заметок, которые там хранятся
Создаются заметки с именем - UUID.формат, имеют заголовок (первая строка заметки и не более 20 символов) и 
тело (никаких ограничений, тело пишется во второй строке)

- Если выбераем посмотреть список заметок, то так же нужно будет выбрать инетересующий нас формат.
При просмотре списка, заметки сразу сортируются по дате от старых к самым новым.
Список показывает нам: имя файла - UUID.формат ; заголовок ; дата и время создания/последнего изменения

В этом же меню нам будет доступен выбор: Фильтровать заметки, Редактировать заметку, Удалить заметку или вернуться в предыдущее меню.
--Фильтровать заметки
    В этом блоге меню, позволит отфильтровать:
        - по дню создания/изменения
        - по месяцу
        - по году
    Список отфильтрованных заметок так же будет отсортирован по дате от страх к новым. 
    При каждого фильтра будет предложено выбрать файл, редактировать его или удалить.

--Редактировать заметку
    Нужно выбрать заметку по имени - UUID.формат. Через меню можно будет изменить:
        - заголовок заметки (не более 20 символов)
        - тело заметки

--Удалить заметку
    Нужно выбрать заметку по имени - UUID.формат.

--Вернуться назад

В общем и целом это краткое описание программы. 
Успешного пользования!
